What is a loop in your own words? Repeating code

What are the different types of loops? How are they different from each other? For Loop repeats a set amount of times, While Loop repeats as long as the condition is true, Nested loop is a loop in another loop, Infinite Loop repeats indefinitely

What do the continue and break commands each do? Break ends the loop, while continue skips to the next loop

What is a list in your own words? A sequence of names/words

There are 2 types of loops. How do you know which to use when using loops? For Loop repeats code a specific amount of times, While Loop continues as long as the condition is true
